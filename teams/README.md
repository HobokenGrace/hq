# **Team Overview**

## **Web Team**

Build and maintain our public-facing websites and internal tools that support church communication and engagement.

### Skills / interests

* JS/TS, frontend or full-stack development
* UI/UX, translating ministry needs into usable tools

### Commitment

* **Meeting**: Tuesdays, 7:00–8:30pm (virtual; in-person on the last Tuesday of each month)
* **Work**: Small async tasks between meetings as availability allows

### Good fit if you

Enjoy building things people interact with and are comfortable learning frontend concepts over time.

---

## **Data & Automation Team**

Build behind-the-scenes tools, scripts, and automations that help staff and volunteers spend less time on admin and more time caring for people.

### Skills / interests

Problem solving, scripting, data workflows, systems thinking (technical depth can grow over time).

### Commitment

* **Meeting**: Tuesdays, 7:00–8:30pm (virtual; in-person on the last Tuesday of each month)
* **Work**: Async, flexible tasks during the week

### Good fit if you

Like logical problems, improving processes, and working mostly behind the scenes.

---

## **Analytics Team**

Support accurate data entry and reporting so the church can make informed ministry decisions.

### Skills / interests

Attention to detail, consistency, basic spreadsheet work; less technical, more process-oriented.

### Commitment

* **Meeting**: Sundays, 2:30–3:30pm (virtual)
* **Work**: Light, recurring tasks tied to weekly rhythms

### Good fit if you

Prefer clear, repeatable tasks and supporting the mission through reliability and accuracy.

---

## **Concierge Team**

Manage email communications and mailing lists to help people stay connected and informed.

### Skills / interests

Organization, communication, following defined processes (technical background not required).

### Commitment

* **Meeting**: No recurring meeting
* **Work window**: Monday afternoon → Tuesday evening (weekly rhythm)

### Good fit if you

Want a predictable, time-boxed commitment without meetings.
